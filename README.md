# Frozendpea

A productivity app which locks the user out from the machine after a set of minutes to
prevent RSI related problems and support healthier work.

## Prerequisites

- Go 1.21+ installed: https://go.dev/dl/
- Go get fyne.io/fyne/v2
- Go get github.com/BurntSushi/xgb

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ikit24/Frozenpea.git
   cd your-repo-name
   ```
2. **Run the application:
   ```bash
   go run .
   ```
## Usage

Once the application is running, you will be greeted with a background picture window when
it is time to take a rest (after 50 minutes by default)
